ZOMATO - ANALYSIS

Hey!

We have Analysed and Visualised the Zomato dataset

of Bangalore Restaurent

ABOUT THE PROJECT

Bangalore is a place of variety of Restaurent and Cuisines. Due to increase in

number of population, new Restaurent were arived. So we decided to analyse the

Top Restaurents, Top favourite dishes, Cuisines.

We have took the Zomato dataset from a website name kaggle. We used python libaries

like Numpy, Pandas and Mathplot for Analyse and visualisation.

ANALYSING THE DATA

From the Zomato dataset, we have Analysed and Visualised the Top Restaurents, Most liked

Dishes, How many people were ordering through online(ZOMATO), How many people had

not used zomato to order, Highest Vote and Highest Rating.

Also, we have did some Data Cleaning. Data Cleaning means removing or replacing some

empty values.
